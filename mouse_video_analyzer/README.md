# マウス操作解析ツール (mouse_video_analyzer)

画面録画などの動画から**マウスポインターの位置**をテンプレートマッチングで追跡し、
軌跡・移動速度・クリック（操作の一時停止）候補・ヒートマップを可視化するデスクトップGUIアプリです。

## セットアップ

```bash
cd mouse_video_analyzer
pip install -r requirements.txt
python3 run.py
```

Python 3.10+ を想定しています（`OpenCV` / `NumPy` / `PySide6` を使用）。

## 使い方

1. 「動画を開く」で解析したい動画ファイル (mp4 / avi / mov / mkv / webm) を開く
2. 「① カーソル範囲を選択」を押してから、プレビュー上でマウスカーソルの周りを
   ドラッグして選択範囲（テンプレート）を指定する
   - **コツ**: カーソルの輪郭線を含む、真っ白/真っ黒などの単色にならない範囲を
     選ぶと精度が上がります（単色の塗りつぶしだけだと誤検出しやすいです）
3. 「② 解析開始」でフレームごとにカーソル位置を追跡（進捗バー表示、バックグラウンド実行）
4. 解析後、以下を確認できます
   - **プレビュー**タブ: 軌跡を重ねた動画をスクラブ/再生
   - **ヒートマップ**タブ: カーソル滞在位置の密度
   - **統計**タブ: 速度の時系列グラフとクリック（操作停止）候補の一覧
5. メニュー「エクスポート」から、軌跡CSV／クリックCSV／ヒートマップ画像／
   軌跡つき動画を保存できます

## 仕組み（概要）

- `app/tracker.py`: 選択したカーソル画像をテンプレートとして `cv2.matchTemplate`
  で追跡します。前フレーム位置の周辺を優先的に探索し、信頼度
  （正規化相互相関スコア）が閾値を下回ったら画面全体を再探索します。
- `app/analysis.py`: フレーム間の移動距離から速度を算出し、
  「移動 → 一定時間静止」というパターンをクリック/操作の候補として検出します。
  また、滞在位置をガウシアンぼかしで積算してヒートマップを生成します。
- `app/gui.py`: PySide6 によるGUI。追跡処理と動画書き出しはバックグラウンドの
  `QThread` で実行し、UIをブロックしません。

## 制限事項

- クリック検出は「移動が止まった」ことを基準にしたヒューリスティックです。
  カーソル画像自体が変化しない限り、実際のマウスダウン/アップを直接検出しているわけではありません。
- 高速に動く・背景に酷似した色のカーソルは追跡精度が落ちる場合があります。
